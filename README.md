# google-reviews-widget

A lightweight, zero-dependency widget that renders Google Reviews on any **static website** — no backend, no server, no framework required.

Built and maintained by [GLS Tech Limited](https://glstech.co.uk) — IT support & web services in County Durham.

---

## Features

- Drop a `<div>` and two `<script>` tags into any HTML page — done
- Light and dark themes out of the box
- Overall rating banner with star score and review count
- Responsive grid (3 → 2 → 1 column)
- Hover animations on cards
- Python sync script to pull fresh reviews from the Google Places API

---

## Quick start

**1. Copy the widget files into your project:**

```
your-site/
├── widget/
│   ├── reviews-widget.js
│   └── reviews-widget.css
└── reviews.json        ← generated by sync_reviews.py
```

**2. Add to your HTML page:**

```html
<head>
  <link rel="stylesheet" href="./widget/reviews-widget.css">
</head>
<body>
  <div id="reviews"></div>

  <script src="./widget/reviews-widget.js"></script>
  <script>
    GoogleReviewsWidget.init({
      url:        './reviews.json',
      container:  '#reviews',
      max:        6,
      theme:      'light',
      showBanner: true,
    });
  </script>
</body>
```

That's it. The widget fetches `reviews.json` at runtime and renders the cards.

---

## Configuration options

| Option       | Type    | Default           | Description                              |
|--------------|---------|-------------------|------------------------------------------|
| `url`        | string  | `'./reviews.json'`| Path or URL to your `reviews.json` file  |
| `container`  | string  | `'#reviews'`      | CSS selector for the mount element       |
| `max`        | number  | `6`               | Maximum reviews to display (0 = all)     |
| `theme`      | string  | `'light'`         | `'light'` or `'dark'`                    |
| `showBanner` | boolean | `true`            | Show the overall rating banner           |

## Syncing reviews from Google

Use the included Python script to pull live reviews from the Google Places API and write them to `reviews.json`.

**Requirements:**
```bash
pip install requests
```

**Set your API key as an environment variable — never hardcode it:**
```bash
export GOOGLE_PLACES_API_KEY=your_api_key_here
```

**Run the sync:**
```bash
python sync/sync_reviews.py \
  --place-id YOUR_GOOGLE_PLACE_ID \
  --output ./reviews.json \
  --business-name "Your Business Name"
```

Find your Place ID at [developers.google.com/maps/documentation/places/web-service/place-id](https://developers.google.com/maps/documentation/places/web-service/place-id).

**Automate with cron (Linux/Mac):**
```bash
# Run every day at 3am
0 3 * * * GOOGLE_PLACES_API_KEY=your_key python /path/to/sync_reviews.py --place-id YOUR_ID --output /path/to/reviews.json
```

> **Security note:** The sync script only reads `GOOGLE_PLACES_API_KEY` from the environment. Never commit API keys to version control. Add `.env` to your `.gitignore`.
